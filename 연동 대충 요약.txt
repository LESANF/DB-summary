1. 기본은 try, catch, finally구문 ( finally는 마지막에 반드시 해야할때 조건추가 )

2. String driverName, url 변수화시켜서 넣음 그외에도 ResultSet rs, Connection con, conn, Statement stmt = null로 선언하고 하면 깔끔해짐

3. 연동시 WEB-INF -> lib -> ojdbc파일 넣는거 필수 안넣으면 에러남.

4.   String driverName = "oracle.jdbc.driver.OracleDriver";     String url = "jdbc:oracle:thin:@localhost:1521:XE"; 이런식으로 변수화 시키고 

Class.forName(driverName);    Connection con = DriverManager.getConnection(url, "BOARD","123"); 다음과 같이 선언해줌 url, id, ps순서 id,ps도 따로 빼서 변수화 해줘도됨 ㅇㅇ

jsp에선 out.print이 웹상에 보여지는 출력문임. 

5. Statement는 데이터베이스에 DML쿼리(삽입, 수정, 삭제, 조회)를 전송할 때에 필요한 객체.

크게 executeQuery 와 executeUpdate라는 메서드를 가지고있음.

매개변수로는 둘다 String으로 들어감. executeQuery는 SELECT(조회)문을 전송할 때, 사용하는 메서드로 ResultSet 자료형의 데이터를 반환함.

ResultSet이란 SELECT문을 실행하여 테이블로부터 얻은 결과를 저장하고있는 저장소라고 알면 되겠다.
